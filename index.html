Introduction

It's great to show your users that they need some more information, have done something right, or done something wrong. This tool allows you a simple style framework to use for general page notifications if you prefer. It also allows a notification system to track what your users are doing, and really shines when using a single-page application and can track the notifications through the app's complete lifecycle.
